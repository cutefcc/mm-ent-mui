## 何时使用
任何需要加载图标的地方

## 引用方式
```render javascript
import {Loading} from 'mm-ent-ui'
```

## 代码演示
> 组件名称： loading图标
```render html
class Demo extends React.PureComponent {
  render() {
    return (
      <div>
        <Loading /> 加载中...
      </div>
    )
  }
}
```
